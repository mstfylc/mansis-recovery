# Mansis Recovery

This private repository is intended to collect sanitized recovery, hardening, and restore materials for the Mansis incident.

Do not commit secrets, raw backups, database dumps, SSH private keys, service-account JSON files, or customer data.

## Contents

- `docs/incident-reports/`: sanitized incident and hardening reports
- `docs/restore-runbooks/`: restore and disaster recovery notes
- `infra/`: sanitized infrastructure templates and notes
- `scripts/`: recovery and audit helper scripts, without embedded credentials
- `evidence/`: sanitized evidence summaries
- `desktop/`: desktop release checksums and notes

## External Archives

Large binaries and backups should stay in Google Drive or object storage. Store only links, checksums, and descriptions here.
